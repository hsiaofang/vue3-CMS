## 技術
- **Vue 3**: 前端框架
- **Vue Router**: 路由管理
- **Pinia**: 狀態管理
- **Element Plus**: UI組件庫
- **Axios**: 處理HTTP請求
- **ECharts**: 圖表庫
- **Mock.js**: 模擬數據工具
- **Less**: 樣式預處理器
- **Vite**: 前端構建工具
- **Unplugin-auto-import**: 自動導入
- **Unplugin-vue-components**: 按需加載
